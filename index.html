Localize by stas

WordPress plugin to easily switch to any localization from GlotPress

This plugin allows you to switch your WordPress installation to use any of the languages available on GlotPress.

Check the WordPress.org project page.

Dependencies

  • WordPress
  • Automatic plugins installation available

Install

Please follow the standard installation procedure for WordPress plugins.
